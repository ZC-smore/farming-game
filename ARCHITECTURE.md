## 二、架构分层

```
┌──────────────────────────────────────────────────────────┐
│  🎮 UI 层 (Vue3 组件)                                    │
│  App.vue → WorldMap / ShopView / InventoryView          │
│          → GameHeader / GameToast / GameModal            │
├──────────────────────────────────────────────────────────┤
│  🧠 状态层 (Pinia Store)                                 │
│  game.ts: coins/exp/level/water/farmPlots/ranchSlots/   │
│           gardenPots/inventory/season                    │
├──────────────────────────────────────────────────────────┤
│  ⚙️ 逻辑层 (Systems — 纯函数)                            │
│  crop/animal/garden 状态机 / season 四季 / inventory CRUD│
├──────────────────────────────────────────────────────────┤
│  📋 配置层 (Configs — 纯数据)                            │
│  crops/animals/flowers/levels/economy                    │
├──────────────────────────────────────────────────────────┤
│  🔌 适配器层 (Adapters)                                  │
│  localStorage 存档 / 广告SDK 接口                         │
└──────────────────────────────────────────────────────────┘
```

**数据流方向**：配置 → 逻辑 → Store → UI  
**不会反向**：UI不直接改配置，只通过Store操作来驱动状态机。

## 四、状态机

### 作物生命周期
```
EMPTY ──[plantCrop(idx,cropId)]──→ PLANTED
                                      │ 有水
                                      ▼
                                   GROWING ──[到时间]──→ MATURE
                                      │                       │
                                      │ 无水                   │ harvestPlot()
                                      ▼                       ▼
                                   暂停生长 ←── 浇水恢复      EMPTY (+作物进背包)
                                                       
MATURE ──[1小时不收获]──→ WITHERED ──[clearWithered]──→ EMPTY
```

### 动物生命周期
```
HUNGRY ──[feedAnimalSlot()]──→ FEEDING ──[到时间]──→ PRODUCING
                                                         │
                                                         │ collectAnimalProduct()
                                                         ▼
                                                        READY ──→ HUNGRY (循环)
```

### 花卉生命周期
```
SEED ──→ GROWING ──[到时间]──→ BLOOM ──[pickFlowerPot()]──→ PICKED
```

                    ┌──────────────┐
                    │   进入游戏    │

              ┌────────────┼────────────┐
              │            │            │
     ┌────────▼───┐ ┌──────▼──────┐ ┌───▼─────────┐
     │  🏪 商店    │ │  🌾 农田     │ │  📦 仓库     │
     │            │ │             │ │             │
     │ 买种子 💰  │ │ 空地→选种子  │ │ 查看物品     │
     │ 买动物 💰  │ │ 缺水→浇水💧  │ │ 多选→卖出💰  │

     │ 升级水桶   │ │ 枯萎→铲除🔧  │ │             │

     └──────┬─────┘ └──────┬──────┘ └──────────────┘
            │              │
            │    种子进背包 │    收获进背包
            └──────────────┘

     ┌─────────────┼─────────────┐
     │             │             │
┌────▼────┐ ┌──────▼──────┐ ┌───▼─────────┐

## 六、游戏循环

```
┌─────────────────────────────────────────────────────┐
│  App.vue onMounted                                  │
│                                                     │
│  setInterval(1000ms) ──→ game.updateAllStates()     │
│    │                                                │
│    ├─ regenWater()        # 1单位/2分钟自动回水     │
│    ├─ updateCropGrowth()  # 遍历36地块，检查生长    │
│    ├─ updateAnimalState() # 遍历9栏位，检查产出     │
│    └─ updateFlowerGrowth()# 遍历花盆，检查开花      │
│                                                     │
│  setInterval(30000ms) ──→ game.save()               │
│    └─ localStorage 自动存档                         │
└─────────────────────────────────────────────────────┘
```
